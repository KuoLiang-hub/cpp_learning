11.6 - Selection Sort:
	Swap smallest with current position.
	[5, 3, 1, 4, 8] =>
	[1, 3, 5, 4, 8] =>
	[1, 3, 4, 5, 8]
runtime: (N^2)/2, O(N^2)

	
11.9 - Insertion Sort:
	loop to swap the one next to current position. i=1, j=i-1 for outter loop.
i=1,j=0
	[5, 3, 1, 4, 8] =>
	[3, 5, 1, 4, 8] =>
i=2,
	j=1
	[3, 1, 5, 4, 8] =>
	j=0
	[1, 3, 5, 4, 8] =>
i=3
	j=2
	[1, 3, 4, 5, 8] =>
i=4
	j=3
	[1, 3, 4, 5, 8]
runtime : n(n-1)/2 , O(n^2)
